writeparams

Syntax:

  • writeparams <dw>
  • writeparams <dw> <scal>
where
  • dw - a delta wavenumber/length shift
  • scal - an intensity scaling constant
Write an ASCII file of the current parameter list to 'outfile'. This command writes a readparams command at the front of the list and a keyboard command at the end. The current scheme for retrieving written lists is to use the "run" command to switch input to the file with the line images; at the end of the list, control is returned to the keyboard.

Example:

open outfile 
writeparams
close outfile
This puts output similar to the following into the file : 
readparams
96/03/26 00:00:03
53246.54863   7.4038  270.59 0.0000 -6  0   L  2132.55
53256.20304   4.7331  214.55 1.0000 -6  0   L  1595.12
53400.42592  23.9398  270.11 0.0000 -4  0   L  6883.25
53460.60134   5.0540  286.48 0.0000 -6  0   L  1541.21
53488.61790  50.7037  263.09 0.0563 -4  0   L 14528.75
end-of-params: keyboard

The columns are:

  1. The wavenumber(wavelength)
  2. The peak intensity
  3. The full width at half maximum
  4. The damping parameter
  5. The number of times the line has been fitted
  6. A code for the hold parameters
  7. The origin of the line:
    • F: Line was found with linelist
    • L: Line has been fitted with filtered lsqfit
  8. Integrated intensity
The data can be read back into Xgremlin using the command run filename
