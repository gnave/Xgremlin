writefile

Syntax:

writefile [xy|xybin] <filename> [< nop> ] [<format>]

Write nop points of the data in 'r' to an output file in ASCII, using a FORTRAN format if given. The FORTRAN format must be enclosed in brackets. If the flag 'xy' is given, data are written as unnormalized x-y pairs. Use flag 'xybin' to write data as binned unnormalized x-y pairs.

Example:

sinc 10. 1024
writefile sinc.out 512 (f8.3)
writes 512 points (half) of the sinc function to the file 'sinc.out', using the FORTRAN format f8.3, together with the Xgremlin statements needed to read it in again with the run command. The file looks like the following: 

info id '                      '
info wref        1.000000
info delw    0.10000000000E+01
length    512
