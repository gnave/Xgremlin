close

Syntax: close < unit name >

This command closes the file attached to unit name (with an open command) so that it can either be re-opened as another unit name or so that the unit name can be attached to another file using open.

Valid unit names are:

  • altdata
  • com
  • datain
  • dataout
  • helpfile
  • infile
  • logfile
  • lineio
  • outfile
  • rawin
  • scratch
  • spool

Example:

close datain

Closes the file attached to the unit datain.

Related commands:
