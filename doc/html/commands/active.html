active

Syntax: active [ < list of line numbers > ]

Where the optional list of line numbers is

  • a list of line numbers separated by blanks
  • a range of line numbers written as l - m
  • no list implies all lines
Activating a line means subtracting the calculated line profile (based on the parameters in the line list) from the actual data in the r-array leaving the residuum in the r-array.

Lines in the internal linelist (from a getlines or findlines command) must be activated before a lsqfit command can be used to fit line profiles to data. All parameters are optimized in the fit unless some of the paramteres of the lines have been fixed by one of the hold... commands. However if all parameteres are held using just hold , active undoes the hold and the lines can be refined using lsqfit. To de-activate a line use the drop command.


Examples:

active 1 3 5 7
activate lines 1, 3, 5 and 7
active 1 -100
activate the first 100 lines in an internal line list (2nd number is -ve)
active
activate all lines in the internal line list
