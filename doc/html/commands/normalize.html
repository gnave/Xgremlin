normalize

Syntax:

  • normalize value
  • normalize value1 value2
  • normalize value1 value2 value3
  • normalize npt1 value1
  • normalize npt1 value1 npt2 value2
  • normalize npt1 value1 npt2 value2 npt3 value3
Parameters:
value
the data value to normalize to
npt
point index in the r array
The contents of the r array are normalized to given values at 1-3 points.

normalize without parameters simply scales the data so that the maximum absolute value becomes 1.0; if a single value is given, the maximum is scaled to that value.

normalize a b divides the data by (a + b*i) so that normalization from one scan can be used on another related scan; normalize a b c divides by (a + b*i + c*i*i).

The last three forms divide by a constant, a straight line or a quadratic to bring the given points to the given values.
