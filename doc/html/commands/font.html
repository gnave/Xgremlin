font

Syntax: font < "small" | "large" > < fontname >

This command is used to change the two fonts for the plotter window. A large font is used for tick labels etc. whereas the small font is used for exponents, the mouse position indicator and such. Any X11 font that is available on a given system or via a font server can be used although for best results fixed width fonts should be used. Proportional fonts usually lead to placement errors of tick labels and other misfits.


Note 1:
The UNIX command xlsfonts can be used to find out what fonts are available on a system. This command prints out a list of all fonts to standard output.

Note 2:
The font used for the text window and button labels can only be changed in the application defaults file Xgremlin in the directory /usr/lib/X11/app-defaults or the .Xdefaults files in home directories. Modify the resource *Xgremlin*font to change the font.

Examples:

font small lucidasans-12
Change the small font of the plotter window to lucidasans-12

font large lucidasans-14
Change the large font of the plotter window to lucidasans-14

font large 9x15
Change the large font of the plotter window to the 9x15 pixel fixed font.
