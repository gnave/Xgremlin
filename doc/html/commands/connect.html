connect

Syntax: connect < nstrt > < nstop >

Replace a section of data in the r array between points nstart and nstop by a straight line connecting the two specified points.

Note:
This command can also be executed interactively by defining the range with two mouse markers (left button) and typing in the < meta > +c plot mode command.
