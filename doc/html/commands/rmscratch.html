rmscratch

Syntax:

rmscratch <n1> [<n2>, .... ]

This command removes the scratch file with file numbers n1, n2, ... Use the command whats saved to find out which scratch files are in use.

Example:

rmscratch 2
will remove scratch file scratch.2 from the current scratch file directory
