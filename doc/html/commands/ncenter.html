ncenter

Syntax: ncenter [ ncentr ]

By default, the position of the center (ncentr) of the interferogram is taken from the header file. This command overrides the default. If no parameter (or 0), find peak from data; if a parameter is given, it becomes the effective center.

The correct centre must be used to determine the phase, otherwise there may be 2*pi phase flips which cannot be fitted by a polynomial.
