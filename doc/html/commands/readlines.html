readlines

Syntax:

readlines <filename> <'old' or 'syn'> [<dw> <smul>]

where:

  • "filename" - the name of a writelines ("old") or standard synthetic spectrum ("syn") line list (without any ASCII lines at top or bottom)
  • dw - a fractional wavenumber(length) shift
  • smul - a line intensity scaling constant
read line parameters from a file, put in new .lin file. Any old .lin file will be saved to the file <filename>.lin.bak

The 'syn' form of the command is the most useful for turning a list of spectral lines into a .lin file. For each spectral line the linelist must contain the following information:

identifier, wavenumber, intensity, width, damping

in the following format:

A15, 2X, F12.5, F10.4, F9.2, F8.4

Examples:

read 
linelist 
writelines x.x
...
...
readlines x.x 'old'
will read the file x.x and produce a new .lin file containing those lines.
