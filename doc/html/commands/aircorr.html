aircorr

Syntax: aircorr p t g h

Apply Edlen's wavelength correction for the refractive index of air (B Edlen; The Refractive Index of Air; Metrologia 2, 71-80, 1966 ). The correction is applied when spectral line data are written out to a line list file in in ASCII format with a writelines command.

The command parameters are:

  • p: spectrometer tank pressure in Torr
  • t: air temperature in degree Celsius
  • g: geometrical correction
  • h: spectrometer tank water vapor pressure in Torr
All command parameters must be supplied.


Related commands:
