dump

Syntax: dump <array> <file name without extension> [<ni> <nop>]

This command creates a pair of header (.hdr) and data (.dat) files and writes the current contents of any one of the r, tr, ffta and phz arrays to the file. The file pair will be in NSO type 6 format. A start index ni and the number of points nop must be specified for the phz and ffta arrays. For the r and tr arrays this is an option and if start index and number of points are omitted the entire array is written to the file.


Example:

dump r spectrum
Creates files 'spectrum.hdr' and 'spectrum.dat' with the current contents of the r array.
