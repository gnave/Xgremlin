dispar

Syntax: dispar [ n ] [ m ]

Dispar displays the line parameters in the internal line list starting at line n and continuing to line m or the last line. If only n is given the parameters for that single line are displayed.

Parameters which have been fixed using 'hold' are coded in column 'H' as follows:

  • 1 = damping
  • 2 = width
  • 4 = sigma
  • 8 = amplitude
  • 16 = all parameters; 'active' undoes this flag, but leaves individual parameter holds intact
If more than one parameter is held, H is the sum of the individual values.

The column 'T' lists the processes the line has been through. The following 'tags' are in use:

  • F: line has been found using 'findlines'
  • L: line parameters have been improved with 'lsqfit'
  • I: line has been inserted by hand
  • G: line was inserted following a 'centre of gravity' ( C+g ) command
  • P: line marks the peak found in a centroid ( C+w ) command
