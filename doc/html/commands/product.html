product

Syntax: product

This command does a vector multiply on r and tr. ie, 

r(i) = r(i) * tr(i)    for i=1, ... ,nop


Examples

The following commands simulate an under-resolved line (Voigt convolved with sinc, which is equivalent to multiplying the Fourier transforms. 
sinc 10. 1024
fourtran
exchange
save 1 'FT of sinc'
voigt 10. 1024
fourtran
exchange
save 2 'FT of Voigt'
exchange
recall 1
product
plot
