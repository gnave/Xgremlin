rewind

Syntax:

rewind <unit name>

<unit name> - name of the Xgremlin i/o unit. (see open)

Acceptable names are:

  • com
  • datain
  • dataout
  • lineio
  • altdata
  • outfile
  • scratch
  • spool
  • logfile

Related commands:

  • open
  • files
  • close
