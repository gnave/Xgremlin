combine

Syntax: combine [ < a > < b > ]

Where

  • a - coefficient for the r array
  • b - coefficient for the tr array

Computes a linear superposition of the data in the r and tr arrays, the result is left in the r array: 

                r(i) = a * r(i)  +  b * tr(i)   for i=1, .. ,nop
The constants a and b both default to 1.0


Example:

combine 1.0 -1.0
will subtract the tr array from the r array
