planck

Syntax:

planck <wstart> <wstop> <delsig> <T>

Calculate the Planck function for temperature <T>, for wavenumbers <wstart> to <wstop>, with stepsize <delsig>.

Examples

planck 10000. 20000. .1 12000.
plot
will plot the Planck function at a temperature of 12000 K between 10000 and 20000 wavenumbers.
