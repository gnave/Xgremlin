printparams

Syntax:

  • printparams [ <dw> ] [ <istrt> <istop> ] [ <widths> ]
  • printparams dw [ <scal> ] [ <istrt> <istop> ] [ <widths> ]
  • printparams <ipage>
  • printparams <istrt> <istop> <ipage>
  • printparams <dw> <scal> <ipage>
  • printparams <dw> <scal> <ipage> <widths>
  • printparams <dw> <scal> <istrt> <istop> <ipage> <widths>
where:
istrt :
starting line number (default = 1)
istop :
stopping line number (default = nol)
dw :
shift positions by dw kaysers ( or nm ) dw can be used alone.
scal :
scale amplitudes by smul. If you want to use scal, you have to put in dw.
ipage :
paging flag (appears to be ignored!) ipage = 0 suppresses paging, really any integer argument here will do this. If istrt and istop are used ipage must appear after them.
"widths":
widths flag - a character string flag to specify splitting the voigt width up into two components, gaussian and lorentzian. Can be any string; one character will do.
This command produces parameter lists in wide (120 column) format suitable for spooling to a printer; for interactive use, 'dispar' is more convenient. To produce a printed linelist, it can be used in conjunction with 'spool', which redirects output that usually goes to a screen to a file instead.

Examples: 

open datain spectrum
read wref             # read in a block of data and
findlines 10.         # find all the lines with intensity> 10.
open spool tmp        # open the spool file
spool                 # toggle the output to the file
printparams
close spool           # close the spool file
screen                # and redirect the output to the screen
