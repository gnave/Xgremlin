noise

Syntax:

  • noise nop [ < amplitude >] [add]
  • noise iseed "init"
Produce nop points of noise in the r array by summing 12 random numbers uniformly distributed in [0,1] and then subtracting 6.0 to give zero mean for every point in the r array. The RMS of the noise signal is 1. If the amplitue is given the noise signal will be multiplied by the specified factor. If the string add is on the command line the noise will be added to the data in the r array.

The second form is used to initialize the noise generator. iseed is an integer seed value.
