deglitch

Syntax: deglitch < list of up to 10 exempt points >


Normally Xgremlin will automatically try to eliminate data glitches from an interferogram before transforming it. This command may be used to specify that the data points in the list should not be treated as glitches in the interferogram and should not be reset to other values before transforming.

Note:
It will only very rarely be necessary to use this command.
