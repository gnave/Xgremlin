recall

Syntax

recall <key> [<npts>] ['tr']

where:

  • key : number of saved buffer
  • npts: number of points to read back (optional)
This command will recall a previously saved stretch of the 'r' or 'tr' array from the scratch file into either the 'r' array (which is the default) or the 'tr' array. If the number of points is not specified all saved points will be recalled.

Examples:

recall 4
read back all data from scratch file scratch.4 
recall 4 tr
read back the contents of file scratch.4 into the 'tr' array 
recall 5 4096
read back only 4k points from file scratch.5
