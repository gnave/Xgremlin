fitpoly

Syntax: fitpoly [ < order of polynomial > ]

A straight line (default) or polynomial of higher order can be fitted to a part of the r array e.g. the background continuum of a single line. 2, 4, 6, 8, ... mouse markers must be set before entering the command. Each pair of markers selects a section of spectrum that is to be included in the fit. The maximum total number of points that can be included is 4096. The fitted polynomial is then plotted on the screen. Two blue mouse markers (set with the middle mouse button) may be set to delimit the range of the plot which is useful for extrapolations.

Example:

fitpoly 3
fit a 3rd order polynomial to the data and plot the fit.

Note:
This command is similar to subpoly but it does not subtract the result of the fit from the r array.
