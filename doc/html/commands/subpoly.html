subpoly

Syntax:

subpoly [<order of polynom>] ["restore"]

A straight line (default) or polynomial of higher order can be fitted to a part of the r array e.g. the background continuum of a line. The continuum is then subtracted from the line. 2, 4, 6, 8, ... markers must be set before entering the command. Each pair of markers selects a section of background that is to be included in the fit.

If the command is used to subtract the background continuum of a line the best approach is to select a range of points to the 'left' and a range to the 'right' of the line of interest. The maximum total number of points that can be included is 4096.

Two blue mouse markers (set with the middle mouse button) may be set to extend or narrow the range in which the polynomial is subtracted from the data. This is useful for extrapolation - use with care.

The subpoly restore command undoes the effect of an immediately preceeding subpoly command.

Examples:

  1. subpoly 3
    fit a 3rd order polynomial to the data and subtract the fit.

    • set 4 red mouse markers and two blue ones
    • stom a : stores all the markers
    • fitpoly 3: to see the result of a Chebychev fit
    • recm a: recalls the markers
    • subpoly 3 : to fit again and subtract
