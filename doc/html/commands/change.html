change

Syntax: change < line number > < line variable > < new value >

This command changes the line parameter values in the internal linelist and updates the residuals in the process. Possible values for the line variables are:

  • amp or int: line intensity
  • sigma: line position / wavenumber
  • damp or dmp: damping parameter
  • width: line width in mK
  • tags: 3 characters of line tags
  • id: 32 characters of new line identification
Zero (0.0) serves as a place holder, keeping the previous value; dmp may be set to 0.0 in spite of this convention by using any negative value. The last two versions are editing commands, to enable certain lines to be tagged with the same character for use by e.g. hold, or to give a line an identification.

Example:

change 129 width 117.32

sets the width of line 129 to a new value of 117.32mK
