wavelimits

Syntax:

wavelimits [ <wavlo> <wavhi> ]
  • wavlo - lowest wavenumber to save (rounded to start of block)
  • wavhi - highest wavenumber to save (rounded to end of block)
This command is used to specify or clear the wavenumber range of a transformed interferogram that is to be written to disk in the following 'real' or 'complex' command. If no wavenumbers are specified, the currently set wavelimits are cleared.

Example:

wavelimits 25000. 30000.
complex myfile
The 'complex' command will write the spectrum (transformed interferogram between 25000.cm^-1 and 30000.cm^-1 to the file myfile.dat (+header).
