plotsize

Syntax:

plotsize <width>x<height> [<orientation>]

sets the width W and the height H of screen dumps in inches. The orientation can either be 'landscape' or 'portrait' - the default orientation is 'landscape'. Screen dumps in landscape mode will be centered on the page, screen dumps in portrait mode will be printed at the top of the page and centered horizontally.

Examples:

plotsize 7x4.5
will set the size of screen dumps to 7 inches wide and 4.5 inches high. The orientation is 'landscape' 
plotsize 4x6 portrait
sets the screen dump size to 4" high and 6" wide. The plot orientation is 'portrait'.
