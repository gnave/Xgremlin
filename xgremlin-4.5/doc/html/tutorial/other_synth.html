Synthetic Spectra

Xgremlin has various commands for creating synthetic spectra. This can be useful if you wish to match a known spectrum with one you have recorded, and has been used successfully by Jon Murray to match the hyperfine structure of the Hg 253.6 nm line to his observations and provide a wavenumber calibration of his spectra.

The basic command is the voigt command, which has the form:

  • voigt wl wg nop
  • voigt w nop ncenter
  • voigt w nop
The first creates a Voigt profile of Gaussian width wg and Lorentzian width wl, setting the number of points to nop. The second form creates a Lorentzian in nop points, centred around ncenter. The third form does the same as the second, but centres the Lorentzian at nop/2.

You can investigate the effect of noise on your spectra with the command
noise nop. This creates a noise record in nop points, which can be added to a synthetic spectra placed in tr using combine. There are various other options in the noise command, mainly concerned with the method of generating the noise record - see the help file for further details.
