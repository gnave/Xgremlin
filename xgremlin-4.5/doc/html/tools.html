Xgremlin Line List Tools

dumplin
Line list data files (.lin files) in Xgremlin are binary files. dumplin is a tool that allows to print out the contents of a line list file outside Xgremlin. To some extent dumplin can also be used to repair some problems with line list files.

mergelin
A program to merge the contents of two separate line list files into a single line list file. This is useful to e.g. combine the results of two linelist commands (with different wavelength ranges) into one file for further analysis such as line fitting.

fits2nso
fits2nso converts a data file in FITS format into a pair of header (.hdr) and data (.dat) files in NSO type 6 format so they can be read by earlier versions of xgremlin or Gremlin.
