convolute

Syntax:
convolute < nrise > < ncut > < label >
convolute < nrise > < ncut > < label > < igw >
convolute < nrise > < nsen > < ampnoise > < label >

The parameters of the convolute command are:

nrise
width (in points) of mask at each end of data
ncut
cut off frequency, frequencies are zero'd above
label
save label of transform of convoluting function
igw
width (in points) of gaussian filter in real space
nsen
frequency where signal = noise
ampnoise
amplitude of noise

The contents of the r array are filtered according to the appropriate parameter specification and the resulting spectrum is convolved with the transform previously formed and labeled with an kernel command or other save. The filter specification is as follows:

nrise ncut label
sharp cut off filter
nrise ncut label igw
Gaussian filter of width igw
nrise ncut label ampnoise label
Voigt optimum filter

The filter specifications (first two parameters) are those discussed in filter . This command is the logical inverse of the equivalent restore command.
