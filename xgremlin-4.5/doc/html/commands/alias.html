alias

Syntax: alias [ < newname > < string > ]

An unfortunately-named command that has nothing to do with aliasing in FT spectra, but is a way to execute a set of several other commands by means of a single word; similar to the alias command in various Unix shells.

The string newname will become the new command; it will be expanded to string in the command line at execution time, but need not be the last command on the line. If no new name is given, this command will simply list the existing aliases.

The string may include several commands separated by semicolons, but only the last one will have access to further parameters in the original command line. An alias may use another alias.

Currently no more than 50 aliases are allowed, no erasure is possible once aliases are defined. newname is limited to 12 characters, string may be up to 80 chars, but the final expansion for all aliases in the line is limited to 256 characters.


Examples

alias sc "sinc 20. 400;plot"
makes sc an alias for sinc 20. 400;plot
alias wu "whats up"
try it out


Note:

Strings containing blank spaces must be enclosed in "quotes" to ensure that Xgremlin interprets them as one single string.
