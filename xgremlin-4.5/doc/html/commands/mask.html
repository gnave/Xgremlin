mask

Syntax: mask nrise [ average ] [ gwidth ]

Multiplies the ends of the data in r by a cosine bell which rises from 0. to 1.0 in nrise points. The bell is split into rising and declining parts separated by (nop - 2 * nrise) points. If a value is supplied for average, the mean value of the data in r is set to average before masking. If the string width gwidthis specified, the data is further multiplied by a Gaussian of width gwidth points.

A mask avoids end effects caused by discontinuities in the value and slopes of the data at the ends. This is important when performing an FFT because the data in r is assumed to be one period of an infinitely periodic function; hence any end discontinuities appear to be part of the spectrum.

"mask 0 0.0" can be used to subtract the mean from the data in the r array.
