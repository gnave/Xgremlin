getlines

Syntax: getlines [ "active" ]

Getlines reads line parameters from the .lin file attached to lineio into the internal linelist. Should the internal line buffer be too small to hold the desired number of line parameters it can be resized with the alloc command. All lines between wref (the start of the buffer) and the end of the buffer are loaded from the .lin file.

active specifies that the lines should be made active (i.e. should be subtracted from the data) after reading from the .lin file.

The default is not to subtract the lines from the data.


Example:

	open lineio k11sh2
	getlines
will attach file 'k11sh2.lin' to the unit lineio and getlines reads in all line list data from the line list file.
