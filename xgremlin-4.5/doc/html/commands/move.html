move

Syntax move < src> < srcindex> < dest> < destindex> [ndata]

Move the contents of the array src, starting at srcindex, to the array dest, starting at destindex. src and dest are r, tr, ffta or phz. ndata is the number of data to be moved (defaults to nop).


Examples:

move r 1 ffta 1
will copy the r array over to the ffta array (nop points)

move r 1 tr 100 100
copies r(1)...r(100) to tr(100)...tr(200)
