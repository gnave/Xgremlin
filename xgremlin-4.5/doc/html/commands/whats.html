whats

Syntax:

  • whats arrayname ilo ihi
  • whats variable name
  • whats saved
  • whats up
This command is used to query the status of the various Xgremlin data arrays and to print out the current values of the requested points in the specified array or variable.

whats saved lists the contents of the scratch files.
whats up will tell you what's up

Array names currently understood are:

  • r
  • tr
  • ffta
  • phz
  • ilo - starting subscript of dump
  • ihi - ending subscript of dump
Variable names currently understood are:
  • nol
  • linum
  • ntop
  • npo
  • ncentr
  • disp
  • uvar
  • intregs
  • realregs

Example:

whats r 1 10
prints out the values of the first 10 elements of the r array. 
whats nop
Prints out the number of points in the current buffer.
