index

Syntax

  • index set <thickness (in mm)> [<uncertainty>]
  • index estimate [<chirp width>]
  • index mask <chirp width> [<fractional bandwidth>] ['fft']
This is a special subset of commands for spectrally resolved Fourier transform refractometry.
index set
Set the thickness of the etalon sample.

index estimate
Estimate the refractive index (incl. uncertainty) from the white light interferogram directly.

index mask
Remove center burst and noise from the interferogram but leave the chirp signals that carry the refractive index information.

Examples

index mask 140 0.8 fft
Remove noise and center burst from an interferogram in the ffta array leaving only the refractive index chirps (with a width of 140 points) and apply a cosine bell to the 'edges' of the chirps. This form of the command would be used mostly in the 'other' field of the phase correction and Fourier transform form where it will be applied after the calculation of the phase correction but before the Fourier transform.
