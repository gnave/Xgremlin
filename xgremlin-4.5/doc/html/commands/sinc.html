sinc

sinc <scale> [<nop>]

  • scale - number of points between zeros
  • nop - total number of points in 'r' array
Create a sinc function centered in the 'r' array. 
                                   sin( pi * x )
	sinc function:  sinc(x) = ---------------
                                       pi * x

Example:

 sinc 20. 400
 plot
Creates a sinc function and plots it afterwards
