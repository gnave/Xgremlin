add

Syntax: add < a > [ < i_first > < i_last > ]

adds a real number a to all points in the r-array in the range specified by the integer parameters i_first and i_last. If the range is omitted the number is added to all points in the r-array.


Examples

add 3.14 10 300
add 3.14 to all elements 10 to 300 of the r-array
add 100.0
add 100.0 to all elements of the r-array
add -5.0
subtract -5.0 from all elements of the r-array
