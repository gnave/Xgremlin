fitlines

Syntax: fitlines smin smax

All lines between smin and smax in the .lin file connected to lineio will be fitted using lsqfit. The parameters for lsqfit are given by the set command:

  • use set "lsqitnum" to set iterations
  • use set readlength to set read length
  • use set "stretch" to set stretch factor (if required)
  • use lsqfit filter if the data is underresolved and it is desired to compensate for this by filtering the data prior to fitting (see lsqfit).
  • and type 'lsqfit quiet' to stop every iteration from being listed.


Note:
This command is essentially a version of lsqfit that operates on a complete data file instead of the data in the r array only.


Example

To find all the lines over intensity smin between wstart and wstop, then fit them using 5 iterations, stretching the data first: 
   open datain 
   set nonorm scale
   linelist smin wstart wstop
   set lsqitnum 5
   set readlength 4096
   set stretch 2
   lsqfit quiet
   fitlines wstart wstop
