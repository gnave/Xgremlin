copy

Syntax: copy [ < wstart > < wstop > ] [ < nstart > < nstop > ]

Copy copies a contiguous block of data specified either by wavenumbers wstart and wstop or point numbers nstart and nstop, from the datain unit to the dataout unit. The default is to copy the entire file.

Note:
A create command is necessary before the copy command if nothing is connected to dataout.

Byte reversal is performed if the data in the input file are in a different format to the standard format on the machine on which Xgremlin is running. The output file is always in machine byte order.
