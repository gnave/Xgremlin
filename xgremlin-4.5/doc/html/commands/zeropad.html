zeropad

Syntax: zeropad <shift>

increases the number of data points in the r array from 'nop' to the next power of 2 and pads the new data points with 0. If shift is requested the unpadded data points are shifted to the centre of the zero padded array.

Example:

zeropad shift

pad data in r array with 0 and shift old data to centre of padded array.
