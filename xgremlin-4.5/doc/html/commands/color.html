color (or colour)

Syntax: color < object > < color-name >

Set (most of the) colors used in the plotting window. This may be useful for people who are (still) using grayscale displays and have problems with the default colors or for those who just cannot bear the default set of colors. The set of available colors depends on the X server and is listed in the RGB color database rgb.txt. Currently the following objects can be assigned a new color (with the default colors in brackets).

plotter-window (white)
The plotting area

plotter-frame (thistle)
Space for labels, line numbers, line info etc.

text (black)
Standard text color

line-marker (goldenrod1)
marked lines

active-marker (PaleVioletRed2)
activated lines

aux-marker (green3)
auxiliary line marker e.g. for newly marked lines

button-1 (red3)
mouse button 1 markers (usually the left mouse button)

button-2 (blue3)
mouse button 2 markers (usually the middle mouse button)

phase (red3)
phase data and phase axis

phase-bad (bisque3)
bad phase points after they have been marked invalid

phase-fit (blue3)
phase correction polynomial

phase-std (SkyBlue1)
phase fit 66% confidence interval

real-data (black)
real part of data in the r array

imag-data (blue3)
imaginary part of data in the r array

buffer-meter (goldenrod1)
indicator of how much of the r array is visible in the plot

Examples:

color line-marker RoyalBlue
will set the color used for marking lines to RoyalBlue.

color plotter-frame Lavender
will set the frame of the plotter window to Lavender
