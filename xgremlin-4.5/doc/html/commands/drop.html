drop

Syntax: drop [ < list of line numbers > ]

The list of line numbers is of the form l1 l2 l3 ... where the elements l are:

  • a line number like e.g. 5
  • a range of line numbers, written as l1 -l2
  • no list implies all lines
Undoes the effects of active. The lines in the internal linelist are added back to the data if they have been previously subtracted using active. This should restore the data to the original state (before active and lsqfit), and the lines can no longer be refined using lsqfit.

Note:
This command only applies for lines where itn < 0.

Examples:

drop 5
restore line number 5

drop
de-activate all lines in the internal buffer.
