hold

Syntax: hold [ < list of line numbers > ] [ "c" ] [< con >]

Parameters:

list
list is of the form l1,l2,l3,..., where elements l are either a line number or a range of line numbers, written as nl1 -nl2 i.e. the first line number is positive and the second line number is negative. No list implies all lines in the internal buffer.

c
any single character, except blank
con
a negtive real number. All parameters of lines in list are to be held at current values; lsqfit will ignore these lines. The optional character argument is a tag for narrowing the list of lines to be held to only those lines in the list which are tagged with the specified character. If the parameter con < 0, the effect of hold is undone and all parameters are fitted. active no longer undoes the effects of hold.
