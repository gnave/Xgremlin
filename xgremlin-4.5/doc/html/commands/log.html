log

Syntax: log [ < n > ] [ < errval > ]

n - base of logarithm. errval - if r(i) < 0.0, this will be substituted for log(r) Log with no arguments takes a natural logarithm of the contents of the r array. An integer argument causes the log to the base n to be taken. The optional parameter errval is substituted for log(r) if r(i) < 0.0.


Example

log 10
calculate the log base 10 of the data in the r array:
        r(i) = log  ( r(i) )     for  i = 1, ... ,nop
	          10
