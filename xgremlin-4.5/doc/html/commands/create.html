create

Syntax: create < filename >

This command creates and opens a new data output file. It must be used on all new files which are to be written to using write or copy . Files attached to the unit name dataout must be closed prior to executing create (see files command). If the file filename exists, the create command will fail and print an error message.


Example:

create foobar
