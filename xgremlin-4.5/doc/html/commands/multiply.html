multiply

Syntax: multiply < factor> [ < ibeg> < iend> ]

factor is a real number which multiplied to all points from ibeg to iend in the r array. If ibeg and iend are omitted the whole r array is multiplied by the factor.

Example:

      multiply 3.14 10 300
