atlas

Syntax:

  • atlas "rewind"
  • atlas "local"
  • atlas scale [ylo yhi] OR
  • atlas scale epsilon ["stretched"]
  • atlas label "up to 50 characters of label"
  • atlas name "name of output file"
  • atlas wstart wstop dispersion ["noquantize"] OR
  • atlas wstart wstop ndat

The command parameters have the following meaning:
rewind
rewinds the plot file without leaving Xgremlin
local
specifies data already in memory, NOT on a file
ylo, yhi
the y plot limits for a linear plot. Default 0.,1.
epsilon
the noise level for loglin; if absent, plot is linear
stretched
halves the log-linear scale to a maximum of 1000
label
up to 50 characters of plot label
name
name of the output file which is the input file for 'psplot'. The default name is 'atlas'.
wstart
starting wavenumber
wstop
ending wavenumber
dispersion
(real number) linear dispersion on plot, in wavenumbers/cm OR
noquantize
if present, top panel begins exactly at wstart
ndat
(integer) number of points plotted (easier for use with the program PLOT)

Write a special data file (unit=PLOT, filename="atlas.data") for the program psplot. For linear plots, set nonorm [val] should be typed prior to using atlas to ensure the whole atlas is on a consistent scale (otherwise each block of 4096 points will be normalized to 1!). It is recommended that val should be the reciprocal of the noise level, which is the rms deviation from the mean determined when the area command is executed on a portion of the spectrum that is free from lines.

Three calls are required to create an atlas plot with the first two preceding the third:

  1. atlas scale [ylow yhigh] OR epsi
  2. atlas label "LABEL NAME"
    (atlas name "name of output file" is optional)
  3. atlas wstart wstop dispersion OR ndat
If a mistake is made in typing these commands, the output file can be overwritten by typing atlas rewind and then re-entering the commands in order.

The program psplot creates a Postscript file from the output created during the atlas commands which may then be printed. Type man psplot at a shell prompt for more information on psplot.
