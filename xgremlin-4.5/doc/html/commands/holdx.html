holdamp, holdwidth, holddamping, holdsigma

Syntax:

  • holdamp [ < list of lines > ] [ < con > ]
  • holdwidth [ < list of lines > ] [ < con > ]
  • holddamping [ < list of lines > ] [ < con > ]
  • holdsigma [ < list of lines > ] [ < con > ]


Parameters:

list
list is of the form l1,l2,l3,..., where elements l are either a line number or a range of line numbers, written as nl1 -nl2. No list implies all lines in the internal buffer.

con
a numeric control parameter:
  • con > 0: lock amplitude/width/damping/sigma of all lines in list at con
  • con = 0: lock all lines in list at current value
  • con < 0: unlock all lines in list
The intensities, widths, damping parameters or wavenumber of all the lines in the list will not be refined by lsqfit, and will be either held at the current value or set to the value con. All other parameters will be fitted. holdx is not affected by active.
