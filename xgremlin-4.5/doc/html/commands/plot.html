plot

Syntax:

  • plot all
  • plot data
  • plot [ <nstrt> <nstop> ] [ <ymin> <ymax> ]
  • plot phase
Plot the data in 'r' within the window nstrt, nstop, ymin and ymax. Defaults are:
ymin,ymax = 0.0,1.0
nstrt,nstop = 1,nop

plot 'all' plots all points in the r array

plot 'data' turns off phase plotting

'plot phase' specifies a special format used to display phase points. See 'phcorr' for a discussion on how to use the phase plot to check that the phase correction has been correctly performed.

Examples

sinc 10. 1024       #Sinc function, normalized to 1.
plot 400 600 -0.2 0.2 # Expands the central lobes a bit
