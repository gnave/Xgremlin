wplot

Syntax:

wplot <w_begin> [ <w_end>] [ nm ]

plots the part of a spectrum between the wavenumbers <w_begin> and <w_end> if that part of the spectrum is in the current buffer.

If the string nm is added to the command the data on the command line must be in secondary units nm.


Example:

wplot 372.296 372.3 nm
plots the part of the spectrum in the current buffer between 372.296nm and 372.3nm
