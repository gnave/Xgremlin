input

Syntax:

input [ m ]
only reads m points from the data file into the ffta array. If m is omitted, all points in the data file are read in.

input center
only process the center points needed for phase.

input inverted
multiply raw interferogram by -1.0
The input command is used to read an interferogram into the ffta array for transforming. If a skip n command preceeded the input command then the first n points of the interferogram will be skipped (i.e. not read into ffta). If the number of points m is specified with the input command only m points will be read from the interferogram (after skipping). The interferogram is checked for glitches and the mean is computed. Use input center to read in the central 32k points and perform a medium resolution Fourier transform in order to find the phase. Use input inverted if the phase is close to +-pi.


Example:

skip 10000           ( skip the first 10000 points )
input 131072         ( read the following 128k points )
