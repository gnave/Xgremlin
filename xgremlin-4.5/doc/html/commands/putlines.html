putlines

Syntax: putlines


Putlines writes line parameters from the internal linelist to the .lin file attached to unit lineio. It is highly recommended that you bind putlines to one of the configurable buttons. Then it is easy to save the lines you are working on regularly when working through a long file buffer-by-buffer.
