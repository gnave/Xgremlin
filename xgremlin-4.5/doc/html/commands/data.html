data

Syntax: data < ni > [ < nf > [ < nstep|ntrans > ]] < value > [ < array > ]

The command modifies the specified array elements replacing them with value starting at point ni and continuing to point nf by nstep increments. The default step is 1. array can be either r, tr ffta or phz. Default array is r.


Examples:

data 1 1000 2.0
sets all points between 1 and 1000 of the r array to 2.0
