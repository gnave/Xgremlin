remove

Syntax:

remove

removes a single line from the internal line list. When removing several lines, with a succession of 'remove' commands, make sure that they are removed with the highest line number first, or you may not remove the line that you want to remove!

NOTE: It is better to use the plot mode command C+d instead of 'remove' because it gives immediate visual feedback.

Examples:

set 'read normalize'
read <wref>
findlines 0.1
dispar
remove 3                      # Removes the third line in the list
dispar
