area

Syntax: area [ < n_start > < n_stop > ]

Where

  • n_start: starting point number (default 1)
  • n_stop: starting point number (default nop)
Find the sum, mean, RMS deviation from the mean and the center of gravity of the data in the r-array between the points n_start and n_stop. The mean is also saved in xreg(9) for later use if desired. If n_start and n_stop are omitted all points in the current buffer are used.

This command can also be executed whilst within the plot window by moving the cursor to the initial position (n_start) and pressing the left mouse button which sets a marker for the starting point number, then mark the stopping point number with a second mouse marker (left button again) and press Meta+a .

Example:

area 2385 2512
calculate sum, mean, etc of data points 2385 to 2512 in the r-array
