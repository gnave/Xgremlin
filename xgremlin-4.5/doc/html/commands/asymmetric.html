asymmetric

Syntax: asymmetric

This command is used only in conjunction with large Fourier transforms of aymmetric interferograms. The command sets a flag to inform Xgremlin that the interferogram is one-sided i.e. it needs to be multiplied by an amertising function before it is transformed. This overrides any information in the header section of data files. The asymmetric command rarely needs to be used interactively.
