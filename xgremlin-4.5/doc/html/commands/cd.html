cd

Syntax: cd < pathname >

changes the current working directory to "pathname". Directory names may be specified in four different ways:

1) absolute directory names
The complete directory name must be typed in, beginning with the root directory '/'.

Example: cd /disk2/home/ulfg/spectra/neon

2) relative to the current directory
Directory names may be specified relative to the current directory. Any directory name that does not begin with '/' is assumed to be relative to the current directory

Examples:

  1. cd neon
    changes to directory /disk2/home/ulfg/spectra/neon if the current working directory (see pwd help entry) was /disk2/home/ulfg/spectra
  2. cd ..
    changes to the parent directory

3) relative to the home directory
Directory names beginning with a '~' (tilde) are relative to the home directory (also supported by POSIX compliant UNIX shells).

Examples:

  1. cd ~
    changes to your home directory
  2. cd ~/spectra
    changes to the directory 'spectra' in your home directory.

4) return to previously visited directory
The special directory name '-' (minus) takes you back to the previously visited directory.

Example: cd -
