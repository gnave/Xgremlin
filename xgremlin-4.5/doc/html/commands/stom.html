stom

Syntax:

stom [<r>] [<b>]

store the positions of a set of red (r) or blue (b) or all (default) markers in a plot into a buffer.

Examples:

  • stom r: stores the parameters of all red markers
  • stom: will store the positions of all markers (default)

NOTE: use the command recm to recall the marker positions
