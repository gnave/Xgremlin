goto

Syntax: goto [<w_center>] [ <w_range > ["nm"] ] [ <line number>] ["next"] ["prev"]

plots a range of w_range wavenumbers centered at w_center if that part of a spectrum is in the current buffer. If w_range is omitted the wavenumber or wavelength range is taken from the current plot. If the the string nm is added at the end of the command the numbers on the command line are converted to wave numbers first before passing them to the program.

If the number of a marked line is given as command parameter this line will be plotted if it is in the current buffer. Subsequent lines can be plotted with the goto next command whereas preceeding lines are plotted with goto prev.

Examples:

goto 32156.
center plot at 32156 cm^-1 without changing the current dispersion of the plot.

goto 312. nm
convert wavelengths to primary units and center plot there.

goto 23
plots the spectrum around line 23. A line list file (.lin) must be open.

goto next
plots the spectrum around line 24 if this command follows the goto 23 command.
